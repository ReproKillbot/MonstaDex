About

Hello, I'm Lauren Blakley, an aspiring web developer and designer. I have graduated in 2018 with a Bachelor of Science Degree in Web Design and Development from Lewis-Clark State Collage. They are also able to even choose whatever attack they want for any monster in their party. I've build this site using Django, the API rest framework and RiotJS.
